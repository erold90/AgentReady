What is WebMCP?

WebMCP (Web Model Context Protocol) is a new W3C standard backed by Google and Microsoft. Released in Chrome 146 Canary (Feb 2026), it lets websites expose structured actions to AI agents running in the browser.

With navigator.modelContext.registerTool(), your site becomes AI-native — agents can discover forms, understand parameters, and complete tasks autonomously. No API server needed.
